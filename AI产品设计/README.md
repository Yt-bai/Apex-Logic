# 💞 AI 产品设计 = 交互可控（好用） + 能力可靠（好准） + 安全可信（敢用） + 成本可持续（能规模化） 

## 先定边界：你要做的是“AI 解决什么问题”

**目标用户 & 场景**：谁在什么时刻用、替代什么原流程？

**任务类型**：生成（Gen）/ 检索（RAG）/ 分类（CLS）/ 排序推荐（Rec）/ 代理（Agent）/ 多模态（MM）？

**成功标准**：提效（省时/省钱）or 增收（转化/留存）or 体验（满意度/完成率）？

**是否必须用 AI**：规则/模板/搜索能否更便宜地解决？
