# SQL INSERT INTO
**INSERT INTO**用于向数据库表中加入新的一行。id列是自动更新的，不需要定义。

```text
INSERT INTO table_name (column1,column2,column3,...)
VALUES (value1,value2,value3,...);
```
这表示，**插入新的一行，column1列的值为value1、column2列的值为value2、、、**

如果有的column没写，也就是没有指定值，那么默认为0。

*注：如果不指定列名也可以，那么value要和表中的列一一对应*

*比如：*
```text
INSERT INTO table_name
VALUES (value1,value2,value3,...);
```

# SQL UPDATE
**UPDATE**用于更新表中已存在的记录。

```text
UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;
```
其中，**WHERE**语句非常非常关键！这规定了哪些记录会被更新，如果漏了，每一行对应column的值都会被统一更新成一样的内容。

---

# SQL DELETE
**DELETE**用于删除表中已有的记录。

```text
DELETE FROM table_name
WHERE condition;
```
*注：把表中所有行全部删除的方法*
```text
DELETE FROM table_name
```
此时表是空的，但是表的结构、属性、索引不变。
