# CREATE TABLE
**CREATE TABLE**用于创建数据库中的表，每个表必须有表名，每个表都由行和列组成。

```text
CREATE TABLE table_name
(
column_name1 data_type(size),
column_name2 data_type(size),
column_name3 data_type(size),
....
);
```
其中，**column_name**是列名，**data_type**是该列中字段的数据类型（要替换成int、varchar等等），**size**是每个字段内容的最大长度。

---

# DROP TABLE
**DROP TABLE**用于删除整个表，整个表结构、属性都不再存在，索引不到。

```text
DROP TABLE [IF EXISTS] TABLE_NAME;
```
删除整个数据库：
```text
DROP DATABASE IF EXISTS my_database;
```
只是清空表，表结构、属性、索引保留：
```text
TRUNCATE TABLE TABLE_NAME;
```
