# SQL SELECT
**SELCET**用于从数据库中选取数据，选取出的数据被存在一个结果表中。

```text
SELECT column1, column2, ...
FROM table_name;
```
与
```text
SELECT * FROM table_name;
```
表示从table_name表中取名为column1、column2的列或者整个表（*）

---

# SQL SELECT DISTINCT
**SELECT DISTINCT**用于返回查询的字段中不重复的所有值。

```text
SELECT DISTINCT column1, column2, ...
FROM table_name;
```
从table_name中选取column1和column2不重复的所有值；（1，x）和（1，y）都会被返回，只有（1,1）会被去重。

---

# SQL WHERE
**WHERE**和SELECT...FROM...搭配使用，选取数据集中满足特定条件的记录。

```text
SELECT column1, column2, ...
FROM table_name
WHERE condition;
```
*注：SQL中用单引号表示文本值，数值不用加引号*

*例如*

```text
SELECT * FROM Websites WHERE country='CN';
SELECT * FROM Websites WHERE id=1;
```
| 运算符 | 描述 |
|------|------|
| = | 等于 |
| <> | 不等于。**注释：** 在 SQL 的一些版本中，该操作符可被写成 `!=` |
| > | 大于 |
| < | 小于 |
| >= | 大于等于 |
| <= | 小于等于 |
| BETWEEN | 在某个范围内 |
| LIKE | 搜索某种模式 |
| IN | 指定针对某个列的多个可能值 |
| AND | 连接以上运算符的两种，同时满足才被选取 |
| OR | 连接以上运算符的两种，满足一种即可被选取 |

# SQL ORDER BY
**ORDER BY**和SELECT...FROM...搭配使用，表示对查询结果按照BY后的字段进行排序。

**不写默认升序，写的话ASC是升序，DESC是降序。**

```text
SELECT column1, column2, ...
FROM table_name
ORDER BY column1, column2, ... ASC|DESC;
```
*例如*
```text
SELECT * FROM Websites
ORDER BY a,b DESC;
```
这表示，先按字段a降序排序，如果字段a相同，再按字段b降序排序。

---

# SQL LIMIT
**LIMIT**也是搭配SELECT...FROM...使用，在SELECT的结果中取前n条（LIMIT n）。

```text
SELECT * FROM student
LIMIT 5;
```
*如果是LIMIT m,n;那就表示从第m条开始,取后面的n条。（不取第m条）*

通常会再搭配**ORDER BY**一起使用，比如取成绩前五名：
```text
SELECT * FROM Websites
ORDER BY grades DESC
LIMIT 5;
```
