## AI产品完整生命周期
### 1.机会识别（Discover）：找到“值得做”的 AI 场景与商业机会
### 2. 立项与方案（Define）：明确目标、范围、路线、资源与风险
### 3. 数据与原型（Prototype）：数据可用性验证 + 快速原型验证价值
### 4. 模型与系统（Build）：训练/微调/提示工程 + 系统工程化
### 5. 上线与灰度（Launch）：评估、A/B、灰度、回滚、监控
### 6. 增长与运营（Scale）：指标驱动迭代、成本优化、体验提升
### 7. 治理与合规（Govern）：安全、隐私、版权、内容治理、审计
### 8. 复盘与演进/退场（Evolve/Sunset）：平台化、能力复用或下线
